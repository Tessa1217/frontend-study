# React-Router 없이 SPA 구현하기

## 목표

- **클라이언트 사이드 라우팅** 직접 구현
- **3개 경로** 지원
  - / -> Home
  - /posts -> Posts 목록
  - /posts/:id -> Post 상세

## 환경 세팅

```
npm create vite@latest spa-no-router --template react-ts
cd spa-no-router
npm i
npm run dev
```
